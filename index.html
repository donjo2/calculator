Help

Here are a few tips to help you use this calculator effectively.
  • To show more functions, go to Menu, then tap Functions
  • Turn on dark theme by tapping Menu, then Dark mode
  • Properly close brackets to ensure accuracy of results
  • To save a value, press the M+ button
  • To recall a saved value, press MR
  • Clear screen with AC or clear each value with CE
  • Keyboard inputs are supported: you can use Backspace to delete each value, Delete to clear all, and Return or Enter key to compute.
